# Line Follower Robot CAD Design Portfolio

## Overview

This repository contains the CAD designs and development files for a Line Follower Robot (LFR) created using Autodesk Fusion 360.

The project focuses on designing a lightweight, compact, and competition-ready robot chassis while ensuring easy assembly, proper component placement, and manufacturability through 3D printing.

## Skills Demonstrated

* CAD Modeling using Fusion 360
* Mechanical Design
* Design for 3D Printing (DFM)
* Component Integration
* Assembly Design
* Engineering Drawing Creation
* Rapid Prototyping

## Project Features

* Custom robot chassis design
* N20 motor mounting system
* Wheel and hub design
* Sensor mounting provisions
* Battery and electronics compartment
* Lightweight structural design
* Modular assembly for easy maintenance

## Software Used

* Autodesk Fusion 360
* Bambu Studio / Cura (for slicing)

## Components Considered

* ESP32 Development Board
* TB6612FNG Motor Driver
* N20 DC Gear Motors
* OE-28 Magnetic Encoder
* IR Sensor Array
* Li-Po Battery

## Repository Contents

* CAD Models (.f3d / STEP / STL)
* Assembly Files

* Design Iterations
